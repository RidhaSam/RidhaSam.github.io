Welcome to My Data Science Portfolio!

This portfolio collects some of my work with data science related topics like data visualization, machine learning, exploratory data analysis, etc.

EDA - Analyzing Past IPL Auctions

Gaining experience with Plotly while exploring how IPL auctions have changed over the years and constructing ML models.
Tools Used: Python (Plotly, BeatuifulSoup, Pandas, NumPy, Scikit-Learn, Statsmodels)

Dash App - Formula 1 Championship Results

In this project, I experimented with the Dash framework for building apps for data visualization. In addition, I used SQL as opposed to pure pandas to gain more experience with SQL. Will continue to work on improving it.
Tools Used: Python (Plotly, Dash, Pandas, Sqlite3), SQL

EDA - Economic Inequality

Praciticing the use of Plotly and building linear regression models.
Tools Used: Python (Plotly, Pandas, Statsmodels)

Djent Song Generator

In this project, I developed a recurrent neural network based on the one presented in this article by Sigurður Skúli to generate songs in the style of "djent." Djent is a subgenre of progressive metal that is generally more focused on forming interesting rhythms rather than melodies. The model was trained on 13 songs converted to MIDI and generates its own music as a new MIDI file.
Tools Used: Python, Keras, TensorFlow, Music21

EDA - Batting and Bowling in Test Cricket

Gaining experience with R while exploring batting and bowling statistics of test match cricketers.
Tools Used: R (R Markdown, ggplot2)

Python Implementations for ML Algorithms

Here, I try to implement common ML algorithms, such as linear regression, logistic regression, and Naive Bayes, in Python in the object-oriented style of Scikit-Learn. I also incorporated different optimizers like batch gradient descent and a Newton's method optimizer.
Tools Used: Python

Kaggle - House Price Regression

My attempt at predicting the sales price of houses in this Kaggle competition . I chose to apply a random forest regressor to the problem, which led me to a mean RMSE of 72687.4 using cross validation on the training data.
Tools Used: Python, Pandas, Scikit-Learn, Jupyter Notebooks
